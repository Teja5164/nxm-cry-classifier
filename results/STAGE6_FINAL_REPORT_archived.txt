 3. INFERENCE PIPELINE MAP
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Input: .wav file (any length, any sample rate)
         │
         ▼
  [1] preprocess_audio()
       • librosa.load() → resample to 22050 Hz, mono
       • Peak normalize to [-1, +1]
       • Centre-crop or loop-pad to exactly 10 seconds (220,500 samples)
         │
         ▼
  [2] extract_mel()
       • n_fft=2048, hop_length=512, n_mels=128
       • fmin=50 Hz, fmax=8000 Hz
       • librosa.power_to_db() → log-mel (128 × T)
         │
         ▼
  [3] normalize_mel()   ← uses norm_stats.json (CRITICAL)
       • Z-score per frequency bin: (x - mean) / std
       • Pad/crop time axis to 431 frames
       • Output: (128, 431) float32
         │
         ▼
  [4] prepare_tensor()
       • Shape: (1, 1, 128, 431) → GPU tensor
         │
         ▼
  [5] load_all_models()
       ┌──────────────────┐  ┌──────────────────┐
       │  BaselineCNN     │  │  CNN+BiLSTM      │
       │  (27.8 MB)       │  │  (18.1 MB)       │
       └────────┬─────────┘  └────────┬─────────┘
                │                     │
       ┌────────┘         ┌───────────┘
       │       ┌──────────┴──────────┐  ┌──────────────────┐
       │       │  CryNet             │  │  SE-ResNet       │
       │       │  (16.0 MB)          │  │  (43.5 MB)       │
       │       └──────────┬──────────┘  └────────┬─────────┘
       │                  │                      │
       └──────────────────┴──────────────────────┘
                          │
                          ▼
  [6] Weighted Ensemble (weights ∝ val_acc)
       • Each model outputs softmax(logits) → (5,) probs
       • Weighted average → renormalize
                          │
                          ▼
  [7] Output
       • Predicted class (argmax)
       • Confidence %
       • Per-class probabilities
       • Reliability: HIGH (≥80%) / MEDIUM (≥60%) / LOW (<60%)
